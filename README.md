# 🌿 InAmigos Foundation NGO Website

A modern, responsive NGO awareness website created for the InAmigos
Foundation internship project.

## 🚀 Project Overview

This project showcases a professional NGO website focused on: -
Education - Environment protection - Animal welfare - Community
development - Volunteer engagement

## ✨ Features

### Pages

-   Home
-   About
-   Projects
-   Impact
-   Volunteer

### Highlights

-   Premium NGO landing page design
-   Glassmorphism cards
-   Responsive layout
-   Smooth animations
-   Impact statistics
-   Volunteer form
-   Modern UI/UX

## 🛠️ Technologies Used

-   HTML5
-   CSS3
-   JavaScript

No frameworks: - React ❌ - Bootstrap ❌ - Tailwind ❌
